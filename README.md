# Mobile Shop (React, 360×1369)

Мобильный каркас магазина под дизайн с картинок:
- Главная: слайдер, категории‑чипсы, сетка карточек (скелеты на загрузке).
- Бэйджи ХИТ/SALE/NEW управляются данными (`product.flags`).
- Поиск: история (LocalStorage) + подсказки результатов по началу слова (case‑insensitive).
- Все изображения помечены как «место для вашей картинки».

## Запуск
```bash
npm i
npm run dev
```
Откройте `http://localhost:5173` (Vite).

## Данные
Ожидается `GET /api/products` (JSON):
```json
[
  {
    "id": 1,
    "title": "Футболка мужская Комары",
    "price": 2800,
    "oldPrice": 3500,
    "discount": 20,
    "imageUrl": "https://.../image.jpg",
    "flags": { "hit": true, "sale": false, "new": false }
  }
]
```
Бэк пока не обязателен: есть фолбэк с демо‑товарами.

## Где подставить свои фото
- Баннеры на главной: `Home.jsx` в `<Slider>` (компонент `.banner`).
- Фото товара: `product.imageUrl` в ответе API.
